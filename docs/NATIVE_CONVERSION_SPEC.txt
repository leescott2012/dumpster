1. Executive Sum m ary
Dumpster is an iOS app for curating Instagram carousel photo dumps. It currently wraps a 
React/Vite web app inside a WKWebView using a custom dumpster:// URL scheme handler. 
Native Swift code handles AI features (Apple Vision photo classiﬁcation via PhotoAnalyzer, 
multi-provider LLM  captions via LLM Service), in-app purchases 
(CreditM anager/PaywallView), and a ﬁle-cabinet settings menu (FileCabinetM enuView). A 
JavaScript bridge (dumpsterBridge message handler) connects the two layers.
The goal is to com pletely rem ove the WKWebView, the dist/ bundle, and the 
DumpsterSchemeHandler, and replace the entire web UI with native SwiftUI views — while 
preserving every existing native feature and adding several pending features that are spec'd 
but not yet built.
The app targets iOS 17.0+ and uses SwiftData for persistence.

11.3. G esture Handling
Single Tap: Detected via .onTapGesture(count: 1). Behavior depends on context:
• If appState.addingToDumpId != nil (pool is in selection mode): toggle selection.
• If in dump context: show the "..." dots menu (set a @State var showM enu = true).
• If in pool context with no active selection: show the "..." dots menu.
Double Tap: Detected via .onTapGesture(count: 2). Always opens the lightbox by setting 
appState.lightboxPhotoId = photo.id. The double-tap gesture must be added before the single-
tap gesture so SwiftUI can disambiguate.
Long Press + Drag (dum p context only): For reordering within a dump. Use 
.onLongPressGesture(minimumDuration: 0.5) combined with .draggable(photo.id) and 
.dropDestination(for: String.self) on the parent container. During drag, apply a scale of 1.05 and 
gold border.

    struct Snapshot {

        let dumps: [(id: String, num: Int, title: String, photoIDs: [String])]
        let timestamp: Date
    }
    
    private var undoStack: [Snapshot] = []
    private var redoStack: [Snapshot] = []

--- page 27 ---
    private let maxSnapshots = 200
    
    var canUndo: Bool { !undoStack.isEmpty }
    var canRedo: Bool { !redoStack.isEmpty }
    
    /// Call BEFORE any mutation to save current state.
    func pushSnapshot(photos: [DumpPhoto], dumps: [PhotoDump]) {
        let snap = Snapshot(
            photoIDs: photos.map { $0.id },
            dumps: dumps.map { ($0.id, $0.num, $0.title, $0.photoIDs) },
            timestamp: Date()
        )
        undoStack.append(snap)
        redoStack.removeAll()
        if undoStack.count > maxSnapshots { undoStack.removeFirst() }
    }
    
    func popUndo() -> Snapshot? {
        undoStack.popLast()
    }
    
    func pushRedo(_ snapshot: Snapshot) {
        redoStack.append(snapshot)
    }
    
    func popRedo() -> Snapshot? {
        redoStack.popLast()
    }
    
    /// Clear history older than 24 hours.
    func pruneOldSnapshots() {
        let cutoff = Date().addingTimeInterval(-86400)
        undoStack.removeAll { $0.timestamp < cutoff }
        redoStack.removeAll { $0.timestamp < cutoff }
    }
